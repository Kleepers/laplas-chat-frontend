# Laplas Chat Frontend

Монорепозиторий для фронтенд приложений Laplas Chat.

## Структура проекта

```
├── client/     # Клиентское приложение
└── admin/      # Админ-панель
```

## Разработка

### Client
```bash
cd client
yarn install
yarn dev
```

### Admin
```bash
cd admin
yarn install
yarn dev
```
